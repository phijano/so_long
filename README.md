## Description

This project is a small 2D game with the minilibx to learn about textures, sprites and tiles.

The objective of the player is to collect all the objects present on the map and exit.

## How to use
```
$> ./so_long map.ber
```

The map will be composed of: 0 for an empty space, 1 for a wall, C for a collectible, E to exit the map, P for starting position of the player and B for monsters.

The W, A, S and D or arrows keys are used to move the main character.

Press the ESC key to close the window and the program.
